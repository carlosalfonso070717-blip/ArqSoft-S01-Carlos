# Catálogo de Videojuegos

## Descripción del Sistema

Este proyecto es una aplicación web para gestionar colecciones personales de videojuegos. Permite a los usuarios agregar, visualizar y organizar información sobre sus juegos favoritos incluyendo título, género, plataforma, año de lanzamiento, descripción e imagen de portada.

La aplicación fue desarrollada utilizando el patrón MVC (Model-View-Controller) con ASP.NET Core y .NET 10. Los datos se almacenan en memoria mediante una lista estática que contiene 12 videojuegos de demostración.

## Funcionalidades Implementadas

- Sistema CRUD básico para gestión de videojuegos
- Visualización de catálogo en formato de tarjetas con diseño tipo grid responsivo
- Filtrado de juegos por género mediante botones dinámicos
- Vista detallada de cada videojuego con toda su información
- Sistema de temas alternables: modo oscuro y modo claro
- Internacionalización completa en español e inglés con cambio dinámico
- Diseño responsivo adaptable a dispositivos móviles, tablets y desktop
- Persistencia de preferencias de usuario (tema e idioma) mediante localStorage del navegador
